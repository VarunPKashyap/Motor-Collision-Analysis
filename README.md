# Motor-Collision-Analysis

Access the web app through this link - http://192.168.0.124:8501
If the above link fails, try diabling firewalls and retry
If the problem still persists you would have to setup the program on your local device.
You would have to first download the csv file and the program, then installstreamlit package, run the command -

streamlit run MotorAnalysis.py

You will be directed to 2 links, select either the local link or the network URL to view the page.
